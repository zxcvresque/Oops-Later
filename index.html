Quick Start Guide

  • 1 Type tasks naturally in the big input bar (e.g., "Call mom tomorrow 3pm high priority")

  • 3 Use the sidebar to switch between Today, Upcoming, Board, and Analytics views
  • 4 Toggle dark mode anytime with the button at the bottom of the sidebar
